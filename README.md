# Campus Crate: Mess & Budget Manager

### Overview
Campus Crate is a Python-based utility designed for students at VIT Bhopal to manage their daily dietary schedules and track personal food expenses outside of the college mess.

### Features
- **Mess Menu Viewer:** Quickly check what's for breakfast, lunch, or dinner based on the current day.
- **Expense Logger:** Input costs for canteen snacks or external deliveries.
- **Budget Summary:** View a running total of your monthly "extra" expenditures.

### How to Run
1. Ensure you have Python 3.x installed.
2. Clone this repository: git clone 
3. Run the application: python main.py

### Usage
Follow the on-screen menu prompts to either view the menu (Option 1) or log an expense (Option 2).

Navigate:

Press 1 to check a specific day's menu.

Press 2 to record a purchase (e.g., "Coffee: 20").

Press 3 to close the program.
